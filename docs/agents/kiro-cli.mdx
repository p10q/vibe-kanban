<Steps>
<Step title="Install Kiro CLI">
  ```bash
  curl -fsSL https://cli.kiro.dev/install | bash
  ```
</Step>

<Step title="Configure AWS credentials">
  Ensure your AWS credentials are configured. You can use AWS CLI:
  
  ```bash
  aws configure
  ```
  
  Or set environment variables:
  ```bash
  export AWS_ACCESS_KEY_ID=your_access_key
  export AWS_SECRET_ACCESS_KEY=your_secret_key
  export AWS_DEFAULT_REGION=us-east-1
  ```
</Step>

<Step title="Start Vibe Kanban">
  Once Kiro CLI is installed and AWS credentials are configured, launch Vibe Kanban:

  ```bash
  npx vibe-kanban
  ```

  You can now select Kiro CLI when creating task attempts.
</Step>
</Steps>

## Features

Kiro CLI provides AI-powered assistance for AWS development tasks including:

- Infrastructure as Code (IaC) development and optimization
- AWS service configuration and troubleshooting  
- Code analysis and security best practices
- Resource management and cost optimization
- Integration with AWS services and APIs

## Model Selection

You can specify which model to use with the `model` parameter in your task configuration. Kiro CLI supports various AWS-optimized models for different use cases.
